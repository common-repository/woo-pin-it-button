Adds a Pin It button on WooCommerce Single Products Images

== Description ==
This plugin adds a Pin It button on WooCommerce Single Products Images.
When hovering on an image in single product page you\'ll see the Pin It Button.

For more information or enquiries, check out plugin page at [TargetIMC](http://targetimc.com/)


== Installation ==
For an automatic installation through WordPress:

    Go to the \'Add New\' plugins screen in your WordPress admin area
    Search for \'WooCommerce Pin It Button by TargetIMC\'
    Click \'Install Now\' and activate the plugin

For a manual installation via FTP:

    Upload the tgt-woocommerce-pinit-button folder to the /wp-content/plugins/ directory
    Activate the plugin through the \'Plugins\' screen in your WordPress admin area

To upload the plugin through WordPress, instead of FTP:

    Upload the downloaded zip file on the \'Add New\' plugins screen (see the \'Upload\' tab) in your WordPress admin area and activate.
